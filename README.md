## Four Core Positions / 四项核心主张

🚫 **Oppose all non-defensive wars** / 反对一切非自卫性战争

🗳️ **Major military actions require public referendum** / 重大军事行动须全民公投

🌍 **5% of global military spending redirected to shared human challenges** / 全球军费5%转用于人类共同挑战

⚖️ **Transnational citizens’ juries to monitor conflict mediation** / 跨国公民陪审团监督冲突调解
